จัดการชุดคำถามได้เอง

สร้าง แก้ไข ลบ ชุดคำถามได้ไม่จำกัด นำเข้า/ส่งออกไฟล์ JSON รองรับทุกวิชา

🏆

ระบบคะแนนและ Feedback

ให้คะแนนตาม Streak และความเร็ว แสดงผลทันทีพร้อมสถิติ ตอบถูก/ผิด/ความแม่นยำ

⏱

จับเวลาแต่ละข้อ

ตั้งเวลา 15 วินาทีต่อข้อ มีแถบ Timer แบบ Visual กดดันได้ระทึก เพิ่มความสนุกในการแข่งขัน

บันทึก Cloud ด้วย Firebase

ชุดคำถามซิงค์อัตโนมัติผ่าน Firestore ใช้ได้ทุกเครื่อง ไม่สูญหาย แม้ปิด Browser

📷

Fallback ปุ่มคีย์บอร์ด

ไม่มีกล้องก็เล่นได้ กด A/D บนคีย์บอร์ดแทนได้ทันที ไม่มีสะดุดในการสอน
